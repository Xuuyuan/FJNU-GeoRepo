# JAVA语言 期末考试感想

- Author: [@Xuuyuan](https://github.com/Xuuyuan)
- Date: 2025/12/26 (2025-2026学年)
- Teacher: 赵珊珊

这门课程的考试安排在了期末周前，这一点还是不错的 —— 这学期可以放个长寒假。
老师之前对考试的题型、分值等表述得很模糊，所以基本约等于看到卷子才知道要考什么。
题型如下：

- 填空题（8题，共15空，每空1分）
- 选择题（10题，每空2分）
- 判断题（10题，每空1分）
- 程序改错辨析题（10分）
- 简答题（3题，每题5分）
- 编程题（2题，每题15分）

【`2025-2026-1打印资料`】文件夹内是我所打印出的全部材料。其中的大部分材料实际上没有用上。根据文件重要性排序，大致如此：

- （有用）`课程教学PPT汇总（黑白）.pdf`：此为本学年课程授课使用的PPT，从学习通中获取。大部分填空题、选择题、判断题从此查找。
- （有用）`大题模板（彩印）.docx`：此为本学年的课堂录音转文字后让AI总结出的大题模板。其中的模板三完美命中了本学年考试的最后一道大题，仅一个得分点有偏差。
- （极少用）`Java语言综合测试（彩印）.docx`：此为本学年最后一次作业的内容，含AI解析。极少数选填可以从里面找到。
- （用不上）`Java语言实验指导书（黑白）.pdf`：此为本学年实验指导书，含AI解答。
- （用不上）`【总纲要】Java复习（黑白）.docx`：此为2023-2024学年学长留下的资料，基本没有改动。
- （用不上）`案例：购物、投币、水仙花数、条件控制（黑白）.pdf`：此为本学年课件中的实验案例，试卷没有考核相关内容。
- （用不上）`学习通1-9章作业讲解（黑白）.pdf`：此为本学年学习通作业讲解。

总而言之，这门课基本不需要怎么复习，但是由于手写代码的缘故，试卷的书写量非常大。我个人答题卡写了三页半，全场无人提前交卷。
